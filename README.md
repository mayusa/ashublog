## 1.需求 Discovery Phase  
a. 这个项目是个人全栈开发项目，目的是开发一个类似于wordpress的中英双语多用户Blog系统。    
b. 文章可以转发到第三方平台（FB,WEIBO,TWITTER, 微信等等）。      
c. 管理员有专用后台，用来进行文章管理，文件管理和数据分析等。    
d. 有数据分析(接入谷歌分析系统)。    
e. V2期有微信公众号关联功能，用户可用微信登录，可以管理同步微信文章和管理微信粉丝      
f. 可以有email订阅功能。
